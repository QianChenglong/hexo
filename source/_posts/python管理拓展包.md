pip提供python的包管理操作

## 安装

-   下载安装脚本[get-pip.py](https://bootstrap.pypa.io/get-pip.py)

-   `python get-pip.py`

-   win下添加`C:\Python27-32\Scripts`到`PATH`

## 使用

-   安装

        pip install PACKET_NAME
