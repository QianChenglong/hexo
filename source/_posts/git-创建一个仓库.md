-   登录[git官网](https://github.com/)

    ![](/img/git-创建一个仓库-01.png)

-   初始化仓库

        git init

-   添加到版本控制

        git add .

-   提交

        git commit

-   设置远程仓库

        git remote add origin git@github.com:QianChenglong/lily.git

-   提交到远程仓库

        git push -u origin master
