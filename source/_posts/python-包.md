# 要点

-   包是一种模块命名空间的组织方法，使用`package.module`方式来访问模块，用来避免模块名冲突

-   python语法上，包是带有`__init__.py`的目录

-   使用`from package import item`导入

    `item`可以是已定义的名字，子模块或子包
