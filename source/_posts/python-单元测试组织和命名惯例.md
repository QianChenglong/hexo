| 名称        | 作用                                         |
|-------------+----------------------------------------------|
| tests       | 目录，位于项目根目录下，用于存放单元测试文件 |
| test_XXX.py | 文件，用于测试某个模块的测试源码             |
| XXXTestCase | 类名，测试用例名                             |
| test_XXX    | 测试方法名                                   |
